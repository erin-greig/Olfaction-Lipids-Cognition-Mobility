# Olfaction-Lipids-Cognition-Mobility

This zip file contains R Scripts used in "Plasma Lipid Metabolites Mediating Olfaction with Cognition and Mobility."

This repository contains files pertinent to linear regression models (LinearRegression.R),
logistic regression models (LogisticRegression.R), 
linear mediation regression models (LinearMediation.R),
and logistic mediation regression models (LogisticMediation.R).

All scripts were run using RStudio v4.3.1. Packages used include "data.table", "tidyverse", "regmedint", "mediation".

Additional codes for figure generation are available upon request.
